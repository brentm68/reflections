automatically stay up-to-date with your remote repository?

    a change on github may overwrite a change on the local repository. 

Describe the differences between forks, clones, and branches.  When would you
use one instead of another?

    a fork is a copy of a repository that continues without effecting the original. 
    clone is a copy of the repo that can effect the original. 
    branch is a split off of the original with a new label and new commit line.
